org.apache.hadoop.hbase

Class TagRewriteCell

  • java.lang.Object
    • org.apache.hadoop.hbase.TagRewriteCell
  • All Implemented Interfaces:
    Cell, HeapSize, SettableSequenceId, SettableTimestamp

    @InterfaceAudience.Private
public class TagRewriteCell
extends Object
implements Cell, SettableSequenceId, SettableTimestamp, HeapSize
    This can be used when a Cell has to change with addition/removal of one or more tags. This is an efficient way to do so in which only the tags bytes part need to recreated and copied. All other parts, refer to the original Cell.

Interface TableProcedureInterface

  • All Known Implementing Classes:
    AddColumnFamilyProcedure, CreateTableProcedure, DeleteColumnFamilyProcedure, DeleteTableProcedure, DisableTableProcedure, EnableTableProcedure, ModifyColumnFamilyProcedure, ModifyTableProcedure, TruncateTableProcedure

    @InterfaceAudience.Private
@InterfaceStability.Evolving
public interface TableProcedureInterface
    Procedures that operates on a specific Table (e.g. create, delete, snapshot, ...) must implement this interface to allow the system handle the lock/concurrency problems.

Class DefaultWALProvider

  • java.lang.Object
    • org.apache.hadoop.hbase.wal.DefaultWALProvider
  • All Implemented Interfaces:
    WALProvider

    @InterfaceAudience.Private
@InterfaceStability.Evolving
public class DefaultWALProvider
extends Object
implements WALProvider
    A WAL Provider that returns a single thread safe WAL that writes to HDFS. By default, this implementation picks a directory in HDFS based on a combination of
    • the HBase root directory
    • HConstants.HREGION_LOGDIR_NAME
    • the given factory's factoryId (usually identifying the regionserver by host:port)
    It also uses the providerId to diffentiate among files.

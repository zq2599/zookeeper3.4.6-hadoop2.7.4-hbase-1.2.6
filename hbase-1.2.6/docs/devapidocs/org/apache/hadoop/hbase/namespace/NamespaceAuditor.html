Class NamespaceAuditor

  • java.lang.Object
    • org.apache.hadoop.hbase.namespace.NamespaceAuditor

  • @InterfaceAudience.Private
public class NamespaceAuditor
extends Object
    The Class NamespaceAuditor performs checks to ensure operations like table creation and region splitting preserve namespace quota. The namespace quota can be specified while namespace creation.

        Check quota to create table. We add the table information to namespace state cache, assuming the operation will pass. If the operation fails, then the next time namespace state chore runs namespace state cache will be corrected.

Class RegionAdminServiceCallable<T>

  • java.lang.Object
    • org.apache.hadoop.hbase.client.RegionAdminServiceCallable<T>

    @InterfaceAudience.Private
public abstract class RegionAdminServiceCallable<T>
extends Object
implements RetryingCallable<T>
    Similar to RegionServerCallable but for the AdminService interface. This service callable assumes a Table and row and thus does region locating similar to RegionServerCallable.

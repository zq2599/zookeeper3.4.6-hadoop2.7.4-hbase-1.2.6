Class HRegionLocator

  • java.lang.Object
    • org.apache.hadoop.hbase.client.HRegionLocator
  • All Implemented Interfaces:
    Closeable, AutoCloseable, RegionLocator

    @InterfaceAudience.Private
@InterfaceStability.Stable
public class HRegionLocator
extends Object
implements RegionLocator
    An implementation of RegionLocator. Used to view region location information for a single HBase table. Lightweight. Get as needed and just close when done. Instances of this class SHOULD NOT be constructed directly. Obtain an instance via Connection. See ConnectionFactory class comment for an example of how.

    This class is thread safe
